## Задачи

### Задача 1: Найти CLAX URL для всех гонок

Не все гонки могут быть на myrace.info. Нужно проверить:
- myrace.info (CLAX формат) — основной источник
- Сайты гонок напрямую (irbisrace.kz, tengriultra.com и т.д.)
- ITRA (itra.run) — для сертифицированных гонок
- PDF/Google Sheets от организаторов — если нет CLAX

**Важно:** Если формат не CLAX — может понадобиться новый парсер (или ручной ввод JSON).

### Задача 2: Спарсить все результаты

Для каждой гонки × каждый год:
1. Скачать и спарсить через `parse_race.py --url ... --save content/races/results/...`
2. Проверить корректность (количество финишёров, наличие клубов, времена)
3. Сохранить JSON в `content/races/results/`

**Внимание к качеству данных:**
- Проверить что клубы спарсились правильно (был косяк когда лишние люди попали в клубы)
- Проверить что имена корректные (кириллица, транслитерация)
- Проверить что все дистанции гонки захвачены

### Задача 3: Расширить races.yaml

Добавить все 7 горных гонок в `content/races/races.yaml`:
- Метаданные (название, тип, локация)
- Дистанции с параметрами (км, набор, высоты)
- Ссылки на editions по годам
- Ссылки на регистрацию 2026

Шоссейные гонки (Almaty Marathon, Half Marathon и т.д.) — добавить минимальные записи (дата + ссылка), без GPX и глубоких карточек.

### Задача 4: GPX файлы дистанций

Для горных гонок — положить GPX в `content/races/gpx/`:
- Alpine Race Skyrunning ✅ (уже есть)
- Остальные — нужно получить от Рената или из открытых источников

GPX нужен для прогнозов (RaceService использует GPX для расчёта через gpx_predictor).

### Задача 5: Проверить бот

После добавления гонок — убедиться что бот корректно показывает:
- Расширенный календарь (не только Alpine Race)
- Результаты и статистику для новых гонок
- Поиск по имени работает

## Результат

- `content/races/races.yaml` — полный каталог
- `content/races/gpx/` — GPX файлы горных дистанций

## Риски

- Не все гонки на myrace.info → ручной ввод или новый парсер
- GPX файлы могут быть недоступны для некоторых гонок → прогноз без GPX (только статистика)
- Качество данных (клубы, имена) — нужна ручная проверка
